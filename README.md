# Royal Zari - Product Image Upload Guide

## 📸 How to Upload Your Product Images

### Step 1: Prepare Your Images
- **File names to use:**
  - `silk-saree-1.jpg` - For your silk saree product
  - `velvet-dupatta.jpg` - For your velvet dupatta
  - `embroidered-kurta.jpg` - For your kurta set
  - `wedding-lehenga.jpg` - For your wedding lehenga
  - `zari-fabric.jpg` - For your fabric product
  - `zari-accessories.jpg` - For your accessories

- **Image requirements:**
  - Size: 300x300 pixels or larger (square images work best)
  - Format: JPG, PNG
  - Quality: High resolution for professional look

### Step 2: Upload Images to GitHub

#### Method A: GitHub Web Interface (Easiest)
1. Go to your repository: [github.com/fahimiqbal0099-star/Royal-Zari](https://github.com/fahimiqbal0099-star/Royal-Zari)
2. Click **"Add file"** → **"Upload files"**
3. Click **"choose your files"** or drag images into the box
4. Upload your product images with the exact filenames above
5. Click **"Commit changes"**

#### Method B: Using Git Commands (Advanced)
```bash
# Copy your images to the images folder
cp /path/to/your/silk-saree-1.jpg /workspaces/Royal-Zari/images/
cp /path/to/your/velvet-dupatta.jpg /workspaces/Royal-Zari/images/

# Then commit and push
git add images/
git commit -m "Added product images"
git push origin main
```

### Step 3: Your Images Will Show Automatically
- Once uploaded, refresh your website
- Placeholder images will be replaced with your real photos
- If image doesn't load, check the filename matches exactly

## 🆘 Troubleshooting

**If images don't show:**
- Check filename is exactly right (case-sensitive)
- Make sure images are in the `images/` folder
- Try refreshing the page (Ctrl+F5)
- Check image size isn't too large (>5MB)
